# AI 编程教学辅助工具 (AI Agent)

Intelligent code comment generation & optimization assistant supporting Python and C++.

## Features

- Upload Python/C++ source files for AI-powered comment generation
- AST-based static analysis for context-aware suggestions
- PPT/DOCX report generation with performance charts
- 50+ teaching examples for both Python and C++

## Tech Stack

- **Backend**: FastAPI + DeepSeek LLM + AST Parser
- **Frontend**: HTML/CSS/JS
- **Reports**: python-pptx, docx, matplotlib

## Quick Start

```bash
cd backend && pip install -r requirements.txt && python main.py
```

Open `frontend/index.html` or use the Claude integration.
