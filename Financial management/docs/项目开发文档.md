# 财务管理系统 - 开发文档

## 1. 项目概述

### 1.1 项目简介
财务管理系统是一个基于 Spring Boot + Vue 3 的前后端分离的记账管理平台，支持用户进行日常收支记录、账户管理、预算规划、账单提醒等功能。

### 1.2 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 后端框架 | Spring Boot | 3.2.0 |
| ORM | MyBatis-Plus | 3.5.5 |
| 安全框架 | Spring Security + JWT | - |
| 数据库 | MySQL | 8.0+ |
| 前端框架 | Vue 3 | 3.4+ |
| 前端构建 | Vite | 5.0+ |
| UI 组件 | Element Plus | 2.4+ |
| 状态管理 | Pinia | 2.1+ |
| HTTP 客户端 | Axios | 1.6+ |
| JDK | Oracle JDK | 17+ |
| 项目管理 | Maven | 3.8+ |
| Node.js | - | 18+ |

## 2. 数据库设计

### 2.1 数据库 ER 图说明

系统包含 6 张核心表：

| 表名 | 说明 | 核心字段 |
|------|------|----------|
| user | 用户表 | username, password, nickname, status |
| account | 账户表 | user_id, name, type, balance, currency |
| transaction | 流水记录表 | user_id, account_id, category_id, type, amount |
| category | 分类表 | user_id, name, type, icon, color |
| budget | 预算表 | user_id, category_id, amount, start_date, end_date |
| bill | 账单表 | user_id, name, amount, due_date, status |

### 2.2 表关系
- user 1:N account - 一个用户拥有多个账户
- user 1:N transaction - 一个用户有多条流水记录
- user 1:N category - 一个用户有多个分类
- user 1:N budget - 一个用户有多个预算
- user 1:N bill - 一个用户有多个账单
- account 1:N transaction - 一个账户有多条流水
- category 1:N transaction - 一个分类有多条流水
- category 1:N budget - 一个分类可有多个预算

## 4. 开发环境搭建

### 4.1 前置条件
- JDK 17+
- Node.js 18+
- MySQL 8.0+
- Maven 3.8+
- IDE (推荐 IntelliJ IDEA)

### 4.2 初始化数据库
```bash
# 登录 MySQL
mysql -u root -p

# 执行初始化脚本
source D:\code\Financial management\db\init.sql
```

### 4.3 启动后端服务
```bash
cd backend
mvn clean install -DskipTests
mvn spring-boot:run
```
后端默认启动在 `http://localhost:8080`

### 4.4 启动前端服务
```bash
cd frontend
npm install
npm run dev
```
前端默认启动在 `http://localhost:3000`

## 5. 项目开发流程规范

### 5.1 分支管理
- `main` - 主分支，保持稳定
- `dev` - 开发分支
- `feature/xxx` - 功能分支
- `fix/xxx` - 修复分支

### 5.2 开发流程
1. 从 dev 分支创建 feature/xxx 分支
2. 在 feature 分支进行开发
3. 完成后提交 MR/PR 到 dev 分支
4. 代码审查通过后合并
5. 发布时合并 dev 到 main

### 5.3 代码规范
- **后端：** 遵循阿里巴巴 Java 开发手册
- **前端：** 遵循 Vue 3 组合式 API + `<script setup>` 风格
- **命名规范：** 类名大驼峰、方法名小驼峰、数据库字段下划线
- **提交规范：** `feat:` / `fix:` / `docs:` / `refactor:` / `test:`

### 5.4 后续扩展方向
- 报表图表统计（ECharts 集成已完成依赖引入）
- 数据导入导出（Excel）
- 账单周期提醒
- 多币种支持
- 预算超支告警
- 数据备份与恢复

## 6. 核心功能模块说明

### 6.1 用户认证
- 使用 BCrypt 加密密码
- JWT Token 鉴权，有效期24小时
- 登录后将 token 存入 localStorage

### 6.2 账户管理
- 支持银行卡、现金两种类型
- 支持账户间转账
- 实时显示余额

### 6.3 流水记录
- 支持按时间、类型、账户、分类筛选
- 分页查询
- 每笔流水关联账户和分类

### 6.4 预算管理
- 按分类设置预算
- 进度条直观显示使用比例
- 超支红色预警

### 6.5 账单提醒
- 设置到期日和提前提醒天数
- 支持标记已支付/未支付
