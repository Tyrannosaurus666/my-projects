### 🤖 Claude

Claude Code 内置的 `/model` 命令只能切换 Anthropic 模型（Opus 4.7 / Sonnet 4.6 / Haiku 4.5），不支持切换到 DeepSeek。

不过 CLAUDE.md 中提到了 `cc-switch` 工具，这应该是这个教师项目的自定义工具，用于在 Claude 和 DeepSeek 之间切换。你可以在终端中尝试：

```
cc-switch deepseek
```

如果 `cc-switch` 不可用，那可能需要通过外部方式（如配置不同的 API endpoint）来使用 DeepSeek，这样就不是 Claude Code 本身的功能了。建议在终端里确认 `cc-switch` 是否可用。
