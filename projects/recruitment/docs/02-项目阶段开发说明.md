**Entity 层 (4 个)**:
- `entity/User.java` — @TableName("user"), @TableLogic, @JsonIgnore on password
- `entity/JobPost.java` — @TableName("job_post"), 5 业务字段 + deadline LocalDateTime

- `entity/InterviewNote.java` — @TableName("interview_note")

**Mapper 层 (4 个)**: UserMapper, JobMapper, ApplicationMapper, InterviewNoteMapper (均 extend BaseMapper)

**DTO 层 (6 个)**:
- `dto/RegisterRequest.java` — @Valid: @NotBlank username + password + role
- `dto/LoginRequest.java` — username + password
- `dto/LoginResponse.java` — token + userId + role + username
- `dto/JobRequest.java` — title @NotBlank @Size(1-100), salaryText, requirements, deadline
- `dto/StatusUpdateRequest.java` — status @NotBlank, content, nextStage
- `dto/UserInfoResponse.java` — userId, username, role

**Service 层 (3 接口 + 3 实现)**:
- `UserService` / `UserServiceImpl` — register (BCrypt), login (JWT签发), getCurrentUser
- `JobService` / `JobServiceImpl` — CRUD + 分页搜索 + 角色权限 (HR只改自己的, admin全局)
- `ApplicationService` / `ApplicationServiceImpl` — 投递(文件上传), 列表(scope=mine|jobId), 状态更新(含面试备注自动创建)

**Controller 层 (3 个 · 11 个 P0 API)**:
- `AuthController` — POST /api/auth/register, POST /api/auth/login, GET /api/auth/me
- `JobController` — GET/POST /api/jobs, GET/PUT/DELETE /api/jobs/{id}
- `ApplicationController` — POST /api/applications, GET /api/applications, PUT /api/applications/{id}/status, GET /api/applications/{id}/notes
