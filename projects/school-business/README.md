# 校园二手交易平台 (School Business)

WeChat Mini Program for campus second-hand trading.

## Features

- Goods listing with images, categories, and search
- Real-time chat between buyers and sellers
- Favorites, reviews, and user profiles
- WeChat Cloud backend (no dedicated server needed)

## Tech Stack

- **Frontend**: WeChat Mini Program (WXML/WXSS/JS)
- **Backend**: WeChat Cloud Functions + Cloud Database + Cloud Storage

## Quick Start

1. Open with WeChat Developer Tools
2. Ensure cloud development is enabled in `app.json`
3. Deploy cloud functions to your WeChat Cloud environment
